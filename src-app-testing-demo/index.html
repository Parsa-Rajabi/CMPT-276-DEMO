CMPT 276 - In-Class Application Testing Activity

Instructions:

  1. 💻 Open Developer Console (Right Click → Inspect → Console Tab) to see test results.
  2. 🔄 Run the starter test cases included in weatherTest.js.
  3. 📝 Complete the following testing tasks in weatherTest.js:

Task List:

  1. Task 1: Write a unit test for formatWeatherData().
    • Sample input: { main: { temp: 30 }, weather: [{ description: "partly cloudy" }] }
    • Expected output: { temperature: 30, description: "partly cloudy" }
  2. Task 2: Write a unit test for convertCelsiusToFahrenheit().
    • Test with inputs:
      • 15°C → Expected 59°F
  3. Task 3: Write an integration test for fetchWeather() using a valid city.
    • Test input: "New York"
    • Expect the response to contain temperature and description properties.
  4. Task 4: Write an integration test for fetchWeather() using an invalid city.
    • Test input: "InvalidCity"
    • Expect the test to handle the error gracefully and assert that an error is thrown.
